  4. 前端技术中台

    本项目是根据公司的需求，共同开发出来中台系统，主要作用是用于提高开发者的效率，无需重复劳动。该项目的前端页面是基于 React 和 TypeScript 技术栈。使用的UI库为Ant Design ，在构建方面使用了Umi 构建的React项目，路由使用了Umi 自带的功能，状态管理则使用Dva 。

    本人负责该项目的部分页面的交互实现及优化，以及对设计图的实现。

    在本项目遇到的难题：本项目展示的数据以及请求的数据次数比较多，例如有个页面会需要每秒请求一次（后端要求），主要是为了更新该服务在服务器的进度，而在前端展示进度条，并且在这个页面，会有多个这样的进度条，因此请求量会比较多。解决方法是：首先，请求应该为请求一次后会等待结果返回，返回的结果只要进度没到100%，就再请求一次，直到请求失败或者进度100%。把返回的结果对应到组件的进度条上显示在前端页面，确保数据及时准确。
